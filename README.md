# Scripture Stickers V1

App web simples para colocar um sticker digital translúcido sobre texto de escritura.

## Como testar no VS Code

1. Abra a pasta `scripture-stickers-v1` no Visual Studio Code.
2. Clique com o botão direito em `index.html`.
3. Escolha `Open with Live Server`.
4. A app abrirá no navegador.

## Funções da V1

- Colar texto de escritura.
- Aplicar o texto à página.
- Usar sticker padrão ou carregar PNG transparente.
- Mover sticker sobre o texto.
- Ajustar transparência.
- Ajustar tamanho.
- Exportar o resultado como imagem PNG.

## Observação

O sticker incluído é provisório para testar a mecânica.
Depois ele deve ser substituído por stickers PNG finais.
